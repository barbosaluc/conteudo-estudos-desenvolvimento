## 1.0 Básico
####  1.1 Algoritmo
Sequência finita de passos bem definidos para resolver um problema ou executar uma tarefa. É a base de todo programa de computador. Um algoritmo deve ser preciso, finito e eficaz.

 Exemplo: o algoritmo de busca binária divide uma lista ordenada ao meio repetidamente para encontrar um valor em O(log n) operações, muito mais eficiente que uma busca linear.
