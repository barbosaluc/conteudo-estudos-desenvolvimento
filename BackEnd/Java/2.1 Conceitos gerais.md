#### Classe
Ela define as características (chamadas de atributos ou campos) e os comportamentos (chamados de métodos), permitindo organizar o código de forma estruturada e reutilizável.

#### Objeto
Cada objeto tem seu próprio conjunto de dados (atributos) e pode realizar ações (métodos) definidas pela sua classe.

#### Pacote (Package)
São mecanismos de organização que agrupam classes e interfaces relacionadas, pense neles como pastas em seu sistema de arquivos, mas para código.

#### Método main()
O método main é o ponto de entrada de qualquer aplicação Java. Quando você executa um programa Java, a Java Virtual Machine (JVM) procura por um método:
`public static void main(String[] args)` para começar a execução.

#### Modificadores de Acesso
- **`public`**: O elemento é acessível de qualquer lugar, dentro ou fora do pacote.
- **`protected`**: O elemento é acessível dentro do mesmo pacote e por subclasses (mesmo que estejam em pacotes diferentes).
- **`private`**: O elemento é acessível apenas dentro da própria classe onde foi declarado.
- (Sem modificador explícito, também chamado de "package-private" ou "default"): O elemento é acessível apenas dentro do mesmo pacote.
