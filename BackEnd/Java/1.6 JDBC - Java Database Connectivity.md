É a ponte oficial do Java que permite que sua aplicação interaja praticamente com qualquer banco de dados relacional.
Para fazer uma consulta ao banco, você sempre seguirá estes passos:

1. **<font color="#ff0000">DriverManager</font>:** É o coordenador que gerencia os drivers de banco de dados. Ele abre a conexão.
2. **<font color="#ff0000">Connection</font>:** É o túnel aberto entre seu código e o banco.
3. **<font color="#ff0000">Statement / PreparedStatement</font>:** É o "mensageiro" que leva o seu comando SQL pelo túnel.
4. **<font color="#ff0000">ResultSet</font>:** É a tabela de resultados que o banco devolve para você ler.
