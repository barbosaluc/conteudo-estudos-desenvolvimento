- <font color="#ff0000">Tipos</font> Primitivos – byte, short, int, long, float, double, char, boolean.
- <font color="#ff0000">Wrappers</font> – Classes equivalentes aos tipos primitivos (Integer, Double etc.).
  As classes wrapper mais comuns incluem:
	- `Integer` para `int`
	- `Double` para `double`
	- `Boolean` para `boolean`
	- `Character` para `char`
- <font color="#ff0000">Strings</font> – Classe imutável para manipulação de texto.
- <font color="#ff0000">Variáveis</font> – Locais, de instância e estáticas.
- <font color="#ff0000">Constantes</font> – Declaradas com ``final``.
  São variáveis cujo valor não pode ser alterado após sua inicialização. Elas são declaradas usando a palavra-chave ``final``.
