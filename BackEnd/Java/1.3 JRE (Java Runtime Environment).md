É o ambiente em tempo de execução Java, com um conjunto de recursos que permite que programas e aplicações Java sejam executadas em um computador.
### O que o JRE Inclui?

O JRE é composto por dois componentes principais:

- <font color="#ff0000">**JVM (Java Virtual Machine):**</font> A máquina virtual que interpreta e executa o **bytecode** Java. Ela é a responsável por traduzir o código para o sistema operacional, garantindo a portabilidade da linguagem.
- <font color="#ff0000">**Bibliotecas e APIs (Application Programming Interfaces):**</font> São as classes e pacotes de código pré-escrito que a JVM usa para executar o programa. Elas incluem a **Java Standard Library**, com funcionalidades essenciais para operações de rede, manipulação de arquivos, interface gráfica (GUI), entre outras.
