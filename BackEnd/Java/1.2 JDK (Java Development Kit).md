É o pacote completo para o desenvolvimento de aplicações Java que contém três componentes principais: 
- <font color="#ff0000">JVM (Java Virtual Machine):</font> A máquina virtual que executa o código Java.
- <font color="#ff0000">JRE (Java Runtime Environment):</font> O ambiente de tempo de execução. Ele contém a JVM e as bibliotecas e arquivos essenciais que o seu programa precisa para ser executado. Ela é o que o usuário final precisa para rodar um aplicativo Java.
- <font color="#ff0000">Ferramentas de Desenvolvimento:</font>
	**`javac` (Java Compiler):** Compila o seu código-fonte (`.java`) em bytecode (`.class`).

	**`java`:** O carregador e interpretador de aplicativos que inicia a JVM para executar o seu programa.

	**`javadoc`:** Uma ferramenta que gera a documentação HTML a partir dos comentários no seu código.

	**`jar`:** Uma ferramenta para empacotar classes e recursos em um único arquivo `.jar` (Java Archive).
